# MiMo Remote

> 🤖 Remote control MiMo Code from your Android phone with voice & video calls

## Features

- 📱 **Real-time Terminal Mirror** — View MiMo Code output on your phone live
- ⌨️ **Remote Input** — Send text and commands from phone
- 🎤 **Voice Call** — Real-time voice, speech→ASR→MiMo Code
- 📹 **Video Call** — Screen sharing + camera
- 🧠 **Memory Browser** — View project memory, checkpoints, task progress
- 🌳 **Task Tree** — Visualize T1→T1.1→T1.2 task hierarchy
- ⚡ **Quick Actions** — /goal, /dream, /distill, Compose one-tap
- 🔐 **E2E Encrypted** — libsodium encryption, server can't read
- 🏠 **P2P Direct** — LAN auto-discovery, no server needed

## Quick Start

```bash
# 1. Install CLI
npm install -g mimo-remote-cli

# 2. Start on computer
mimo-remote start

# 3. Install Android APK & scan QR code
```
